About the App

A simple dog browsing app built with a real backend.

Dog Breed Finder is made for people who want a casual way to explore dog breeds and keep track of dogs they like. The project is also designed to show the required web development skills for INST377.

Target users

The main users are dog lovers, students, and people who are curious about dog breeds. The design keeps the app simple so users do not need instructions to start browsing.

Target browsers

The app is designed for current desktop versions of Chrome, Edge, Firefox, and Safari. It should also work on mobile browsers, but the main target is desktop use.

How the technology fits together

The frontend never calls the Dog API or Supabase directly. It uses Fetch API to call the Node/Express backend. The backend handles the external Dog API requests and the Supabase database requests. This keeps the project organized and matches the final project requirement that frontend data loads go through the backend.
