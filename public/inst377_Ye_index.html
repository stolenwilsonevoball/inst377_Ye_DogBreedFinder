Find dog breeds and save your favorites.

This web app lets users load random dog pictures, browse by breed, and save favorite dogs with a short note.

Open Dog Finder About the project

Project requirements covered

  • Node.js backend
  • Supabase database
  • External Dog API
  • Chart.js and Swiper.js
  • Vercel deployment ready

Browse dogs

Use the external Dog CEO API through the backend to get random dogs and breed-based images.

Save favorites

Save breed names, dog image links, and notes to the Supabase database.

View patterns

The app creates a simple chart showing which breeds were saved the most.
